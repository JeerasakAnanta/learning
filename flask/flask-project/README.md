    - # Flask Web Application 
    - Date: 2024-05-20 
    - ## Description
    - This is a simple Flask web application that serves a basic HTML page.
    - ## Requirements

    ```
    uv add Flask
    ```

    - create a virtual environment and install the required packages:
    ```bash
    uv venv 
    ```
    - use virtual environment
    ```bash
    source .venv/bin/activate
    ```

    - ## Run the Application 
    ```bash
    uv run main.py
    ```
