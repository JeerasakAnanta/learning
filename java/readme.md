# Java
## how to run java in commanline  on ubuntu  
- Open Project 
```bash  
cd ~/Documents/JavaPrograms
```
- Compile  
```bash 
javac YourProgramName.java
```
- Run the compiled Java program
```bash
java YourProgramName
```

## Example 
- s
```bash 
javac HelloWorld.java
```
- run 
```bash 
java HelloWorld
```
- display 
```
Hello, World!
```
